# Cowpea_Phenomics
Data hosting for a study on cowpea phenomics using sUAS-based RGB images

This repository contains the dataset associated with the manuscript:
“Small Unmanned Aircraft Systems (sUAS)-Based Growth Trait Analysis for Biomass Prediction and Trait Stability in Cowpea [Vigna unguiculata (L.) Walp.]”
Submitted to The Plant Phenome Journal
